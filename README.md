# Lea Mae Contiñeta - Portfolio

A modern, responsive portfolio website showcasing web development and design skills.

## Features

- **Hero Section**: Eye-catching introduction with animated elements
- **About**: Personal background and professional summary
- **Skills**: Visual representation of technical proficiencies
- **Services**: Web development, graphic design, and digital marketing
- **Responsive Design**: Fully optimized for mobile and desktop
- **Modern Styling**: Dark theme with orange accent colors
- **Smooth Animations**: Elegant transitions and hover effects

## Files

- `index.html` - Main HTML structure
- `styles.css` - Complete styling and responsive design
- `profile.jpg` - Profile photo

## How to Use

1. Replace `profile.jpg` with your actual profile photo
2. Update the content in `index.html` with your information
3. Customize colors in `styles.css` using the CSS variables
4. Host on GitHub Pages (automatic for this repository)

## Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --accent-orange: #ff6b35;  /* Change this color */
}
```

### Update Social Links
Modify the social links in the hero and footer sections in `index.html`

## View Online

Your portfolio is automatically hosted at: `https://leamae112603.github.io`
